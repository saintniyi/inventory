Description / About
===================
A multi-project ASP.NET Core template for a RESTful API-based MVC Web App, built with Products and Suppliers models. It follows the Unit of Work pattern and Separation of Concerns architecture across projects like InventoryModel, InventoryData, InventoryAPI, InventoryShared, and InventoryWeb.
